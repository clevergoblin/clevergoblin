Centos7的Hugo搭建和使用

Feb 22, 2021
1 min read

1.下载Hugo

1.wget https://github.com/gohugoio/hugo/releases/download/v0.81.0/hugo_0.81.0_Linux-64bit.tar.gz

2.到Github直接下载hugo_0.81.0_Linux-64bit.tar.gz

2.解压

tar -zxf hugo_0.81.0_Linux-64bit.tar.gz

sudo cp ./hugo /usr/local/bin

查看是否安装成功

hugo version

3.Hugo的使用

新建网站

./hugo new site firstsite

添加主题，进入刚才创建的站点

git init
git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

主题可以到Hugo主题去找

然后把主题配置到配置文件中

echo 'theme = "m10c"' >> config.toml

接下来就可以新建文章编写了

hugo new post/blog.md

文章开头如下： title: “Blog” date: 2021-02-22T12:12:52+08:00 draft: false

可以在编辑器中打开进行编写，编写完成之后把draft改为false(为true视为草稿，将不会被部署)

4.启动Hugo服务器

hugo server

在启用草稿的情况下启动服务器使用下面的命令

hugo server -D

这样就可以在本地看到自己的博客了

5.将个人博客部署到Github

在Github上新建仓库，执行下面的命令会生成public目录，命令中的地址更换为自己的

hugo --theme=m10c --baseUrl="https://clevergoblin.github.io/" --buildDrafts

进入public目录，添加远程地址，推送博客

git init
git add .
git commit -m "个人博客"
git remote add origin https://github.com/clevergoblin/clevergoblin.github.io.git
git push -u origin master

此时就可以在线上看到自己的博客了，比如：https://clevergoblin.github.io
